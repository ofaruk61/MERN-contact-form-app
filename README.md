# Edu-MERN-Contact-Form-App

#### This project is a Contact Form application coded with MERN technologies.

## Libraries used on the NodeJS side;
1. ExpressJS
2. Mongoose
3. dotenv
4. cors
5. nodemailer

## Libraries used on the React side;
1. react-bootstrap bootstrap
2. react-router-dom
3. react-toastify
4. react-helmet
5. axios
